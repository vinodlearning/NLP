## 📋 **Quick Start Commands**

### **Build the Project**
```bash
cd OracleADFChatbotSystem
mvn clean compile
```

### **Generate Model Files**
```bash
mvn exec:java -Dexec.mainClass="com.company.contracts.service.ApacheOpenNLPModelGenerator"
```

### **Run Tests**
```bash
mvn test
```

### **Package for Deployment**
```bash
mvn package
```
